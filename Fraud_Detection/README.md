# Credit Card Fraud Detection

This project detects fraudulent credit card transactions using Machine Learning.

## 📌 Description
The goal of this project is to identify whether a transaction is fraudulent or legitimate based on transaction data.

The dataset contains anonymized features (V1–V28), along with Time and Amount.

## ⚙️ Features
- Random Forest Classification model
- Streamlit web application
- Detects fraud in real-time
- Uses transaction features like Time, Amount, and V variables

## 🛠️ Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Streamlit
