# Movie Rating Prediction

This project predicts movie ratings using Machine Learning.

## Features
- Predict movie rating based on:
  - Year
  - Duration
  - Votes
  - Genre
- Random Forest Regressor model
- Streamlit web application

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Streamlit

## How to Run

```bash
streamlit run movie_app.py
