# CODSOFT Machine Learning Internship Projects 🚀

This repository contains all the tasks completed during the CodSoft Machine Learning Internship.

---

## 📌 Projects

### 🌸 1. Iris Flower Classification
Predicts the species of iris flowers using machine learning.

### 🚢 2. Titanic Survival Prediction
Predicts whether a passenger survived the Titanic disaster.

### 📊 3. Sales Prediction
Predicts sales based on advertising budget (TV, Radio, Newspaper).

### 🎬 4. Movie Rating Prediction
Predicts movie ratings based on features like genre, duration, votes, etc.

### 💳 5. Credit Card Fraud Detection
Detects fraudulent transactions using classification techniques.

---

## 🛠️ Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- Streamlit
